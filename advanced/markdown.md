# Custom Prompts

{% hint style="info" %}
自定义提示词可主动干预问题、答案、领域标签生成的效果。
{% endhint %}

例如，在下面的自定义提示词中，我们：

* 通过自定义全局提示词要求必须使用英文
* 通过自定义问题生成提示词要求问题必须保持精简
* 通过自定义答案生成提示词要求答案必须风趣幽默
