# Question Management

{% hint style="info" %}
问题构造完成后，可对问题进行过滤和修订，可提升后续数据集的生成质量。
{% endhint %}

<figure><img src="../.gitbook/assets/image (34).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
建议在这个模块删除质量较低的问题（比如跟文献的作者、标注等不相关的问题），避免后续构造出一些质量较低的数据集，并自定义添加好缺失的问题。
{% endhint %}
